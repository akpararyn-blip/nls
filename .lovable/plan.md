## 5. Страница Интернет — изменения

### 5.1 Калькулятор скорости
В блоке «Рассчитать скорость с учетом рабочих мест»: при клике на «Рассчитать» комментарий/пояснение должен раскрываться (state `open=true` по умолчанию после расчёта; рендерить раскрытым, не collapsed).

### 5.2 Новая карточка тарифа «Интернет для МСБ»
В блоке тарифов добавить карточку:
```
Интернет для МСБ
(для малого и среднего бизнеса)   ← мелким текстом
Скорость до 30 Мбит/с
10 000 ₸ / мес
Акционный тариф 15.06.2026 — 31.08.2026
[Оставить заявку]
```
Без галочек/фич. Кнопка открывает модалку LeadForm с предзаполненным комментарием (см. п. 5.4).

### 5.3 Мобильный фильтр-переключатель тарифов
Над сеткой тарифов на мобильных (`@media max-width: 768px`) — segmented control:
`Все | до 100 Мбит/с | до 10 000 Мбит/с`
- Должен помещаться в ширину без переноса (компактные подписи, `white-space:nowrap`, гориз. скролл скрыт).
- Карточки фильтруются по скорости. Карточка МСБ (30 Мбит/с) попадает в «до 100 Мбит/с».
- На десктопе фильтр скрыт, видны все карточки.

### 5.4 Форма + Telegram: запись выбранного тарифа
- `LeadForm` принимает новый необязательный проп `tariffLabel?: string` (например `"до 30 Мбит/с"` или `"до 100 Мбит/с"`).
- При наличии — поле «Комментарий» предзаполняется текстом вида:
  - обычный тариф: `Интересует тариф: до 100 Мбит/с`
  - МСБ: `Интересует тариф: до 30 Мбит/с (акционный тариф МСБ)`
- На странице Интернет поле комментария **остаётся редактируемым** (на других страницах поведение не меняется).
- В `submitLead` / тексте телеграма: добавить отдельную строку `Тариф: <label>`, для МСБ — `Тариф: до 30 Мбит/с — АКЦИОННЫЙ ТАРИФ МСБ ⚡` (заметная пометка). Кладётся в секцию «Данные о заявке».

### 5.5 CTA «Консультация по доп. услугам» (моб.)
Под блоком «Дополнительные решения для офиса» — компактный CTA-блок, видимый только на мобильных.
- Кнопка «Консультация».
- По клику раскрываются 4 кнопки услуг: **Wi‑Fi для гостей / Видеонаблюдение / Монтаж локальных сетей / IT аутсорсинг**.
- Клик по услуге открывает модальную форму, в комментарий подставляется `Интересует услуга: <название>` (редактируемо), и в телеграм идёт строкой `Услуга: <название>`.

### 5.6 Блок «Решения дата-центра» — добавить недостающие
Текущий блок дополнить, чтобы перечислялись все позиции из двух категорий с ссылками:
- **Облачные решения**: Виртуальный дата‑центр (`/iaas`), VPS/VDS (`/vps`), Объектное хранилище S3 (`/object-storage`), Облачное хранилище (`/cloud-storage`).
- **Услуги дата‑центра**: Размещение сервера (`/colocation`), Аренда стойки (`/colocation-full`), Аренда сервера (`/dedicated`).

## 8. Переключатель периода (1/6/12 мес.)

Глобальная правка CSS (используется на нескольких страницах: dedicated, iaas, vps, calculators):
- Контейнер-подложка переключателя получает более контрастный фон (более тёмный/насыщенный токен), чтобы было видно границу группы кнопок.
- На мобильных — `flex` с `min-width:0`, кнопки уменьшаются по padding/font-size, `white-space:nowrap`, помещаются в ширину экрана без переноса.

## Технические детали

**Затрагиваемые файлы:**
- `src/components/nls/Header.tsx` — десктоп-spacing, моб. язык-toggle, floating menu button.
- `src/components/nls/MobileNav.tsx` — pill-переключатель языка, active-page подсветка, авто-раскрытие группы.
- `src/styles/nls.css` — все CSS-правки (spacing, floating btn, hero-image hide, period-switcher, mobile tariff filter, CTA блок, активные стили).
- `src/routes/internet.tsx` — карточка МСБ, моб. фильтр тарифов, открытый комментарий калькулятора, моб. CTA доп. услуг, расширенный блок DcServices, проброс `tariffLabel` в формы.
- `src/components/forms/LeadForm.tsx` — проп `tariffLabel`, предзаполнение комментария, флаг редактируемости.
- `src/lib/submitLead.ts` + `src/services/telegramApi.ts` (формирование текста) — строка `Тариф:` / `Услуга:` в телеграм-сообщении.
- `src/routes/dedicated.tsx` — disabled-логика кнопки Заказать.
- `src/components/nls/Modals.tsx` — при необходимости проброс `tariffLabel` через openLeadModal.

**Не трогаем:** routing/USE_INTERNAL_ROUTING, phone-validation, /spam, GTM, бизнес-логику отправки (кроме добавления строк в текст).
