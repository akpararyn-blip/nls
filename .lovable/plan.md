
## Тест синхронизации Lovable ↔ GitHub

**Цель:** убедиться, что изменения, сделанные в Lovable, автоматически попадают в ваш GitHub-репозиторий.

### Что я сделаю
1. Заменю содержимое `src/routes/index.tsx` (сейчас там placeholder Lovable) на простую тестовую страницу:
   - Заголовок: **NLS**
   - Подзаголовок: **Синхронизация с GitHub работает ✅**
   - Минимальный аккуратный стиль (центрированный блок, белый фон, тёмный текст)
2. Сохраню изменение — Lovable автоматически закоммитит и запушит его в ваш подключённый репозиторий.

### Как вы проверите, что всё ок
- В превью справа появится новая страница с текстом "NLS — Синхронизация с GitHub работает".
- В вашем GitHub репозитории появится свежий коммит от Lovable, изменяющий файл `src/routes/index.tsx`.

### Что дальше
После проверки вы сможете просто писать мне в чате задачи ("добавь секцию услуг", "поменяй заголовок", "сделай форму контактов" и т.д.) — каждое изменение будет автоматически уходить в GitHub.
